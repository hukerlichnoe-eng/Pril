# 🚂 Бортовой помощник машиниста БЧ

Мобильное приложение для локомотивных бригад Белорусской железной дороги (БЧ). Работает на смартфонах под управлением Android.

---

## 🛠 Основной функционал

* **📈 Динамический профиль пути:** Визуализация рельефа в реальном времени на основе данных GPS.
* **🔊 Голосовой информатор:** Автоматическое речевое предупреждение машиниста **строго за 500 метров** до приближения к нейтральным вставкам и ограничениям скорости.
* **📑 Электронный маршрут ТУ-3:** Быстрый расчет общего рабочего времени смены, времени следования пассажиром и чистого времени на посту.
* **💾 Обмен маршрутами (JSON):** Возможность сохранять (экспортировать) записанные перегоны и загружать (импортировать) файлы конфигурации участков от других машинистов.

---

## 📱 Как правильно использовать в кабине локомотива

1.  **Разрешения Android:** После установки приложения обязательно зайдите в *Настройки телефона -> Приложения -> Помощник Машиниста -> Разрешения* и выдайте доступ к **Местоположению (Всегда / Точное геопозиционирование)**.
2.  **Размещение смартфона:** Металлическая обшивка электровозов/тепловозов сильно экранирует сигнал. Для стабильной работы GPS положите телефон как можно ближе к лобовому стеклу кабины.
3.  **Старт:** Откройте приложение во вкладке «Профиль» и нажмите красную кнопку **«АКТИВИРОВАТЬ ИНФОРМАТОР НА ЛИНИИ»**. Как только поезд наберет скорость более 2 км/ч, начнется отсчет метров и ведение по перегону.
